# fukui-kanko-reservation
こちらが日本語版READMEです: [README.ja.md](README.ja.md)

福井県芦原温泉エリアの予約データの可視化。

## デモ
https://code4fukui.github.io/fukui-kanko-reservation/

## 機能

- 阿波羅温泉エリアの総予約データを表示
- 都道府県別予約数の内訳を提供 (ベータ版)
- 予約曲線の可視化を含む (ベータ版)

## データ/API
この プロジェクトは 福井県観光連盟が提供する 以下のAPIからデータを取得しています:
- 最新のホテル情報
- 最新の予約サマリー
- 最新の都道府県別予約サマリー 
- 最新の予約カーブデータ
