# あわら温泉エリア 宿泊予約状況

このウェブサイトは、福井県のあわら温泉エリアの宿泊予約状況を可視化したものです。福井県観光連盟が提供しているオープンデータを活用しています。

## デモ
https://code4fukui.github.io/fukui-kanko-reservation/

## 機能
- 宿泊予約の総合的な推移グラフ
- 都道府県別の宿泊予約状況（β版）
- ブッキングカーブの表示（β版）

## データ・API
本プロジェクトでは以下のオープンデータを使用しています:

- あわら温泉エリア宿泊施設データ
- あわら温泉エリアの宿泊予約状況（総合・都道府県別）
- あわら温泉エリアのブッキングカーブ
