# fukui-kanko-reservation

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Reservation data visualization for the Awara Onsen area in Fukui, Japan.

## Demo
https://code4fukui.github.io/fukui-kanko-reservation/

## Features
- Displays total reservation data for the Awara Onsen area
- Provides a breakdown of reservations by prefecture (beta)
- Includes a booking curve visualization (beta)

## Data / API
The project fetches data from the following APIs provided by the Fukui Prefectural Tourism Federation:
- Latest hotel information
- Latest reservation summary
- Latest reservation summary by prefecture
- Latest booking curve data
